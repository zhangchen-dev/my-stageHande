# 预设测试任务

本目录包含可导入的预设测试任务。

## 使用方法

1. 在测试平台页面，点击右上角的「导入预设任务」按钮
2. 从下拉菜单中选择要导入的任务
3. 任务将自动导入到你的任务列表中
4. 选择该任务，点击「开始测试」即可执行

## 预设任务说明

### 发票录入演示测试（通用版）

这是一个通用的演示流程测试模板，包含以下步骤：

1. **打开发票录入演示页面** - 访问演示URL
2. **等待页面完全加载** - 等待5秒确保页面加载完成
3. **点击「开始演示」按钮** - 使用AI识别点击开始按钮
4. **等待演示引导气泡出现** - 等待2秒
5. **检查是否有地图提示** - 如有「收起演示地图」提示则点击收起
6. **按指引流程点击完成演示** - 使用AI跟随演示指引

## 自定义预设任务

创建JSON文件放在 `public/preset-tasks/` 目录下即可被自动加载。

### 步骤类型

- `goto` - 访问页面，需要填写 `value` 为URL
- `click` - 点击元素
- `fill` - 填写表单，需要填写 `value` 为输入内容
- `wait` - 等待，需要填写 `value` 为毫秒数

### 执行策略

- `selector` - 使用精确选择器
- `ai` - 使用AI识别（推荐）
- `auto` - 自动选择
